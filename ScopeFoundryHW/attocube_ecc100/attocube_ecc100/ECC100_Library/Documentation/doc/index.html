ECC DLL

The ecc.dll is a library that allows custom programming for the ECC100 piezo controller.ECC100 devices can be searched by ECC_Check, connected by ECC_Connect and released by ECC_Close functions. It is possible to handle multiple devices simultaneously. the ECC_Connect function gives a unique handle for the selected device.

Functions that allow set and get operations are named ECC_control... . A boolean parameter (Bln32 set) determines the way which is meant.
